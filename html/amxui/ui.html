Accordion
This is an accordion. Useful for displaying more data in a limited space.

Set a height on the box, and the script does the rest. If a section has overflowing content, it will automatically be made scrollable.

Like any other container, it can have images, buttons, inputs and other elements.

Modal windows

Modal windows can be used for prompts, dialogs, alerts, notifications and more.

There are 2 main types, normal and minimal. A modal can have a header, body, and/or footer, or just a body.

Modals can be animated when displayed (triggered) just by adding a class.

Clicking on the modal overlay or a button with class .close within the modal will close it.
